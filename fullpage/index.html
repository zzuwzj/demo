First Screen

这是一个使用JQuery插件fullPage.js开发的全屏滚动网站

fullPage.js is A simple and easy to use jquery plugin to create fullscreen scrolling websites (also known as single page websites or onepage sites).

Second Screen

  • 支持鼠标滚动
  • 支持前进后退和键盘控制
  • 多个回调函数
  • 支持手机、平板触摸事件
  • 支持 CSS3 动画
  • 支持窗口缩放，窗口缩放时自动调整
  • 可设置滚动宽度、背景颜色、滚动速度、循环选项等等
